# Hello World

A tiny demo extension that says hello and shows a live greeting inside Frontier.

Hello World is the friendly first thing to install when you want to see what a Frontier extension feels like. It adds its own tab and a sidebar section that greet you, let you set a short note, and show a small status line that updates on its own in real time — a gentle, no-stakes way to watch an extension come to life in your workspace. Nothing here touches your work; it's a self-contained little hello you can install, play with, and remove any time.

<!-- screenshot: Hello World tab showing the greeting, the editable note, and the live status line -->

## Features

- A Hello World tab with a greeting and a short note you can edit
- A live status line that refreshes by itself, so you can see the extension staying in sync
- A sidebar section and a one-key command to open it
- A welcome tile so it's one click away from a fresh workspace

## Install

Install Hello World from the **Extensions → Marketplace** tab in Frontier: find Hello World, click Install, and it's added to your workspace in one click (Frontier verifies the download before installing). No setup or configuration needed.
